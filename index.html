Instalación en 3 Pasos

Listo para usar en menos de un minuto.

01

Descarga el Archivo

Elige la herramienta que necesites arriba y haz clic en "Descargar para Windows" para obtener el archivo ZIP oficial.

02

Descomprime el ZIP

Haz clic derecho sobre el archivo descargado y selecciona "Extraer todo". Guarda la carpeta donde prefieras.

03

¡Ejecuta la Aplicación!

Abre la carpeta extraída, ejecuta el archivo `.exe` con el logo de MXP y comienza a trabajar de inmediato.
